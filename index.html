Html ¿Qué es? y su funcion

¿Qué es HTML?


Es el código utilizado para estructurar y desplegar una página web y sus contenidos. podriendo ser párrafos, una lista con viñetas, o imágenes y tablas de datos, etc. Para poder realizar dichas opciones requiere de unas etiquetas que seran explicadas más adelante.



Funcion del HTML


El lenguaje html opera en base a marcadores escritos a partir de las cuales se cifra la estructura y orden interno de una página web. Dicho código fuente hace de ADN de la página Web, diciéndole al navegador de dónde obtener los recursos para su representación y en qué orden. Y siguiendo el código al pie de la letra. Para ello html opera en base a un conjunto de componentes, como son:

  • Elementos. Los ladrillos básicos del lenguaje html, sirven para representar el contenido y sus atributos, así como marcar los parámetros del propio lenguaje.
  • Atributos. Las especificaciones respecto a valor, color, posición, etc. de los elementos incorporados en el código. Por lo general consisten en una serie de instrucciones lógica o numéricas.
